### Brackets 常用的快捷键整理：



`Ctrl+Shift+H`: 可以呼出与关闭文件树

`Ctrl + E`:          快速预览/编辑 css样式/javascript函数/取色选择

`Ctrl + +/-`:       放大缩小编辑区字体大小

`Ctrl + 0`:          重置编辑区字体大小 

`Ctrl + Alt + P`:  打开即时预览功能

`Ctrl + / `:         行注释

`Ctrl + Alt + /`:  块注释

>注：css代码、html代码注释时只能使用块注释快捷键
